## hslog

Prolog (its pure subset) interpreter in Haskell, written just for fun.

### Features

- [x] Basic implementation
- [ ] Backtracking
- [ ] Non-determinism
- [x] REPL
- [x] List notation
- [x] Infix notation
- [ ] Arithmetic operators
- [ ] Cut symbol
- [ ] User-defined operators
- ...

---

### Usage

Build:

```console
$ stack install
```

Launch REPL:

```console
$ hslog-exe [filename]
```

Launch tests:

```console
$ stack test
```
